# 🚀 Job Application Tracker (Full Stack MERN Project)

A full-stack web application designed to help users efficiently track and manage their job applications through different hiring stages. 
And also for Employer to efficiently post jobs, track applicants

## 📌 Project Overview

The **Job Application Tracker** allows users to:

* Add new job applications
* Track application status
* Edit and update job details
* Delete applications
* View all applications in a structured dashboard

This project demonstrates full-stack development using **MERN stack (MongoDB, Express, React, Node.js)** with REST API integration.

## 🚀 Installation & Setup

### 1. Clone the repository
git clone https://github.com/Nabina2002/Job_Application_Tracker__.git


### 2. Backend setup

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend setup

```bash
cd frontend
npm install
npm run dev
```
